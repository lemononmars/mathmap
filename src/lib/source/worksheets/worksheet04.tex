\item \textbf{พีชคณิตของลิมิต} 

กำหนด $a,c$ เป็นค่าคงตัวแล้ว
\[ \limx{a} c = \blank, \qquad \limx{a} x = \blank \]

จงหาค่าของลิมิตต่อไปนี้ โดยการเขียนเครื่องหมายลิมิตทุกขั้นตอน จนกว่าจะคำนวณลิมิต

\end{tasks}
